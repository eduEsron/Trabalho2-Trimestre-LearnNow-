HTML

Tags de Texto:

As tags de texto servem para adicionar diferentes
tipos de texto em seu documento HTML.
São elas:

  • Tags de titulo: <h1>, aumentando seu nivel até <h6>
    (quanto maior o nivel, menor o tamanho do texto)
  • Tag de parágrafo: <p>Insira aqui o texto</p>
  • Tag para Itálico: <p><i>Insira aqui o texto</i></p>
  • Tag para Negrito: <p><b>Insira aqui o texto</b></p>
  • Tag para Quebra de Linha: <br>
  • Tag para Texto pré-formatado: <pre>
  • Tag para criar uma linha horizontal: <hr>

Tags de Mídia:

As tags de mídia servem para inserir
tipos diferentes de mídia em seu documento HTML.
São elas:

  • Tag de imagem: <img> src="" alt=""</img>
  • Tags de Vídeo: <video> <source src="" type="" > </video>
  • Tags de Audío: <audio> <source src="" type="" > </audio>
  • Tag para incorporar outras páginas: <iframe> width="300" height="300" </Iframe>

Tags de listas e tabelas

  • Lista não ordenada: <ul> </ul>
  • Lista ordenada: <ol> </ol>
  • Elementos da Lista: <li> </li>

HTML semântico

tem como objetivo descrever o significado do conteúdo presente em documentos HTML:

  • Cabeçalho: <header> </header>
  • Tag de navegação: <nav> </nav>
  • Seção principal: <main> </main>
  • Rodapé: <footer> </footer>

CSS

Posicionamento

As propriedades position servem para definir como um elemento é
posicionado na página:

  • position: static; O elemento é posicionado de acordo com o Fluxo Normal do documento
  • position: relative; É deslocado em relação a si mesmo com base nos valores de
    top right bottom, e left
  • position: absolute; O elemento é removido do fluxo normal do documento e
    nenhum espaço é criado para o elemento no layout da página.
    O elemento é posicionado em relação ao seu ancestral posicionado mais próximo
  • position: fixed; sempre permanece no mesmo lugar, mesmo que a página seja rolada
  • position: sticky; é posicionado com base na posição de rolagem do usuário.

Cores e bordas

  • color: #; define uma cor para o texto do elemento
  • background-color: define uma cor de fundo para um elemento;
  • border-style: ; define um estilo de borda para o elemento
  • border-radius: ; define o raio da borde

Layouts:

  • Layout de flexbox: layout que utilizado o display:flex; para transformar os elementos
    em flexíveis. Quando os elementos são dispostos como itens flexíveis, eles são dispostos
    ao longo de dois eixos. Além disso é unidimensional, ou seja, trabalha com uma dimensão de
    cada vez, seja linha ou coluna.
  • Layout grid: Diferente do flexbox esse tipo de layout pode trabalhar com duas dimensões ao
    mesmo tempo. Esse layout utiliza o display: grid. a propriedade transforma todos os
    filhos diretos do contêiner em itens de grade.
